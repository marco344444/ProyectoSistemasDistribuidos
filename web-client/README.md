# Web Client (React + TypeScript)

Cliente web del sistema distribuido con arquitectura MVC (controllers, views, services, models).

## Requisitos

- Node.js 18+
- npm
- Backend visual ejecutandose en `http://localhost:8080`

## Ejecutar en desarrollo

```powershell
npm install
npm run dev
```

Aplicacion disponible en `http://localhost:5173`.

## Build

```powershell
npm run build
```
